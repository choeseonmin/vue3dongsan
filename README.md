## 들어가며
### 1. 프로젝트 소개

vue3에 대한 구성과 문법을 공부하기 위해 간단한 원룸 매매 사이트를 구현해보았습니다.

### 2. 프로젝트 기능

프로젝트의 주요 기능은 다음과 같습니다.
- **홈페이지 -** 데이터바인딩, 컴포넌트
- **모달창 -** 이벤트핸들러  
- **정렬 -** sort함수
